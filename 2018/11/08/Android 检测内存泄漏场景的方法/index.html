dump文件是指进程的内存镜像

StrictMode开发调试阶段使用

Android Memory Monitor

Android Studio自带，生成非标准的.hprof文件，然后进行分析。

MAT（Memory Analyzer Tools）

MAT是Eclipse中的插件，可以独立下载使用。可以现将as中生成的.hprof文件转换成标准的.hprof文件，然后再进行分析。

用的最多的就是 Histogram 功能

Memory Usage

Memory Usage是as中自带的功能

也可以使用adb shell命令来获取指定package name的内存镜像

adb shell dumpsys meminfo [package name]

LeakCanary

LeakCanary 在检测内存泄漏的时候会阻塞主界面,但是这时候阻塞肯定是必要的，因为此时必须要挂起线程来获取当前堆的状态。
