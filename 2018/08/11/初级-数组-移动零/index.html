移动零

发表于 2018-08-11 | 分类于 LeetCode

题目描述

给定一个数组 nums, 编写一个函数将所有 0 移动到它的末尾，同时保持非零元素的相对顺序。

例如， 定义 nums = [0, 1, 0, 3, 12]，调用函数之后， nums 应为 [1, 3, 12, 0, 0]。

注意事项:

  • 必须在原数组上操作，不要为一个新数组分配额外空间。
  • 尽量减少操作总数。

解题思路

该题的关键在于取出数组中所有有效的关键元素，其余位置补0，而不需要交换操作。
